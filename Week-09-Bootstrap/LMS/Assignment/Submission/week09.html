Week 09

Week 09 is about bootstrap. Bootstrap is a CSS library that is used to accomplish a layout that will be responsive accross multiple screensizes. It is very simple to implement, the only hurdle to get over is learning all the syntax that is required to accomplish a responsive layout. Everything has a class that has to be added to the element tag you are working on. If you use the correct syntax and add the correct classes in order. You will be awarded amazing results. In this example of my Week 09 coding assignment. My task is to create a mmulti-page website, that accomplishes that. I have to use the bootstrap CSS library and use all the classes together to create a responsive website. One that will work on large screens, phone screens, and everything in between screens. This is a very fun challenge. I have completed most of the requirements. The menu on the top middle area, called the NavBar, is completely responsive across diferent screen sizes. I have chosen to use the -lg class, because to me it made the website look nicer. As I have tried to have the video look better across different screen sizes.
