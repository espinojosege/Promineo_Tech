Week 09 Bootstrap assignment was about installing Bootstrap to our project. Bootstrap is a framework of CSS. It is very powerful in that it can change the way items are displayed dynamically. I really enjoyed this project, it is creative watching things change depending on where you are looking at it. I am currently running live server on my iphone and it works pretty well.
